# NotebookLM インフォグラフィック作成プロンプト: Claude 利用料金について

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: スクエア（正方形）— 内容がコンパクトなので正方形で十分
- **詳細度**: Concise（簡潔）— シンプルな情報なので短く
- **スタイル**: Professional（料金情報は信頼感のあるデザインで）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude の使用量を確認しよう

【対象者】Claude を使い始めた初心者。使いすぎを防ぐために確認方法を知ってもらう。

【構成】ヘッダー、2つのコンテンツブロック、フッターの4ブロック構成。コンパクトに1画面。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「使用量を確認しよう」
   - サブタイトル: 使いすぎ防止のためにチェック！

2. 知っておくべきこと（制限の説明）
   - アイコン: ⏱️ タイマー
   - Claudeサブスクプランには使用量の制限がある
   - 制限の単位: セッションごと、週ごと
   - 「使いすぎると一時的に制限がかかることも」と補足

3. 確認方法（ステップ形式）
   - アイコン: 📊 グラフ
   - Claude アプリ（Web版でもOK）で確認できる
   - 手順: 左下のアイコン → 設定 → 使用量
   - 3ステップを矢印で繋いだフローで分かりやすく表示
   - 「定期的にチェックしよう！」と補足

4. フッター
   - 「ハンズオン中も使用量を意識しておきましょう」

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、背景 #F5F0EB（ライトベージュ）、補足 #4A90D9（ブルー）
- 確認手順の「アイコン → 設定 → 使用量」を矢印付きの3ステップフローで表現
- シンプルで見やすく、要点だけが伝わるミニマルなデザイン
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
```
