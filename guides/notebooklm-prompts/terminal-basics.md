# NotebookLM インフォグラフィック作成プロンプト: ターミナルの基本操作

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— 上から順に読み進める構成
- **詳細度**: Standard（標準）
- **スタイル**: Sketch Note（手書き風で親しみやすく、初心者の不安を和らげる）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】はじめてのターミナル操作ガイド

【対象者】ターミナルを触ったことがない完全初心者。怖がらずに使えるよう、優しいトーンで。

【構成】ヘッダー、4つのコンテンツブロック、フッターの6ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「ターミナル入門ガイド」
   - サブタイトル: エンジニア気分を体験しよう！
   - 補足: マウスはほぼ使えません。キーボードで操作します

2. 覚えておきたいキー操作（3つだけ！）
   - ↑キー: 前のコマンドをもう一度使える
   - Control + C: 処理を中断する（※Commandではない！）
   - 赤い×ボタン: ウィンドウを閉じて終了してもOK
   - 「Control + C」を特に目立たせてください（重要マーク付き）

3. 注意ポイント（2つ）
   - ⚠️ Enterは慎重に！: 怖いコマンドも実行できてしまう
   - ⚠️ 半角/全角に注意！: 通常は半角モード。日本語のときだけ全角に
   - 警告色（黄色やオレンジ）で注意を引くデザインにしてください

4. 便利ワザ（フォルダの開き方）
   - 方法①: フォルダをDockのターミナルアイコンにドラッグ＆ドロップ
   - 方法②: 「cd 」と入力 → フォルダをターミナルにドラッグ＆ドロップ → Enter
   - ドラッグ＆ドロップの動きが伝わるような矢印や図解を入れてください

5. まとめブロック
   - 「困ったら Control + C で中断！」を大きく表示
   - 「これだけ覚えればClaude Codeも使える！」

6. フッター
   - 「次のステップ: Claude Codeを起動してみよう」

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、背景 #F5F0EB（ライトベージュ）、注意色 #E8A838（イエローオレンジ）
- キー操作はキーボードのキーキャップ風のデザインで表現してください
- 注意ポイントは黄色〜オレンジの背景で目立たせてください
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 手書き風のイラストやアイコンで、初心者が怖がらない雰囲気にしてください
```
