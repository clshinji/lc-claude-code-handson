# NotebookLM インフォグラフィック作成プロンプト: スラッシュコマンド

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— コマンド一覧を見やすく縦に配置
- **詳細度**: Concise（簡潔）— コマンドリファレンスなので短く
- **スタイル**: Professional（チートシート的な参照資料に最適）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code スラッシュコマンド早見表

【対象者】Claude Code を使い始めたばかりの初心者。手元に置いて参照できるチートシート風に。

【構成】ヘッダー、3つのグループブロック、フッターの5ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「スラッシュコマンド早見表」
   - サブタイトル: 「/」を押すと候補が出る！↑↓で選択
   - 補足: Claude Codeへの特別な指示コマンド

2. よく使うコマンド（メインカラーで目立たせる）
   - /clear → 会話をリセットする
   - /exit → 終了してターミナルに戻る（/quit でもOK）
   - /model → AIモデルを切り替える
   - /config → 設定を確認する
   - /usage → 使用量を確認する
   - 各コマンドはモノスペースフォント、コマンド名と説明を横並びで表示

3. もしものときのコマンド（オレンジ〜黄色の注意色で区別）
   - esc キー → 前の画面に戻る
   - Ctrl + C → 操作を中断する
   - /resume → 前のチャット履歴から再開
   - /login → ログイン
   - /logout → ログアウト
   - 「困ったときはまずこれ！」というラベルを付ける

4. 操作のコツ（グレー背景の補足ブロック）
   - 「/」を入力すると候補一覧が表示される
   - ↑↓キーで選択してEnterで決定
   - 困ったら Ctrl+C で中断すればOK

5. フッター
   - 「この表を手元に置いて、迷ったら見返そう！」

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、注意色 #E8A838（イエローオレンジ）、補足 #6C757D（グレー）、背景 #F5F0EB（ライトベージュ）
- コマンド名はダーク背景（#1A1A2E）にライト文字のコードブロック風で統一
- コマンド名と説明を2カラムの表形式で整列させ、見やすくしてください
- 「よく使う」と「もしものとき」のグループを色で明確に区別
- タイトルは8語以内、ヘッダーは5語以内、各コマンド説明は8語以内
- ホワイトスペースを十分に確保し、チートシートとしての視認性を最優先
- 印刷しても見やすいシンプルなデザインにしてください
```
