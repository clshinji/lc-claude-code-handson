# NotebookLM インフォグラフィック作成プロンプト: Claude Code を使うときのコツ

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— 4つのコツを上から順に読む構成
- **詳細度**: Standard（標準）
- **スタイル**: Sketch Note（手書き風で親しみやすく、コツが記憶に残りやすい）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code を上手に使う4つのコツ

【対象者】Claude Code を使い始めた初心者。すぐに実践できるコツを覚えてもらう。

【構成】ヘッダー、4つのコツブロック、フッターの6ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「上手に使う4つのコツ」
   - サブタイトル: これだけ覚えればClaude Code マスター！

2. コツ①: 指示は具体的に書く
   - アイコン: 🎯 ターゲット
   - ❌ 悪い例: 「いい感じにして」→ 何をすればいいか分からない
   - ✅ 良い例: 「自己紹介ページを作って。名前は○○、趣味は△△」
   - ポイント: 具体的なほど良い結果に！
   - ❌と✅を赤と緑で色分けして対比を分かりやすく

3. コツ②: 一度にたくさん頼みすぎない
   - アイコン: 📋 ステップ
   - ❌ 悪い例: 「あれもこれも全部やって！」
   - ✅ 良い例: 1つずつお願いする。大きな作業はステップに分ける
   - ポイント: 小分けにした方がうまくいく！

4. コツ③: やり直してOK！
   - アイコン: 🔄 リトライ
   - 思った結果と違ったら「やっぱり○○に変えて」でOK
   - 何度でもやり直せるのがClaude Codeの良いところ
   - ポイント: 失敗を恐れずに試そう！
   - 安心感のある緑色背景で

5. コツ④: 困ったらClaude Code自身に聞く
   - アイコン: 💬 吹き出し
   - 「このエラーはどういう意味？」と聞ける
   - 「次は何をすればいい？」と聞ける
   - ポイント: Claude Codeは優秀な相談相手！

6. フッター
   - まとめ: 「具体的に、1つずつ、何度でも、困ったら聞く！」
   - 4つのキーワードを横並びで強調

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、背景 #F5F0EB（ライトベージュ）、良い例 #28A745（グリーン）、悪い例 #DC3545（レッド）
- ❌悪い例は赤背景、✅良い例は緑背景で、対比を一目で分かるように
- 4つのコツに番号（①②③④）を大きく表示して、順番が明確に分かるように
- 各コツのアイコンを左端に配置して視覚的にリズムを作る
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 手書き風の温かみのあるデザインで、初心者が気軽に実践できる雰囲気に
```
