# NotebookLM インフォグラフィック作成プロンプト: Claude で提供されるアプリの違い

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ランドスケープ（横長）— 3つのツールを横並びで比較しやすい
- **詳細度**: Standard（標準）
- **スタイル**: Bento Grid（比較レイアウトに最適）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claudeで使える3つのツールを比較しよう

【対象者】ITに詳しくない初心者〜中級者。専門用語は避け、分かりやすい言葉で。

【構成】ヘッダー、3つの比較ブロック、まとめブロック、フッターの6ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「Claude 3つのツール比較」
   - サブタイトル: あなたに合った使い方を見つけよう

2. ツール①: Claude（AIチャット）
   - アイコン: 💬 チャットの吹き出し
   - キャッチコピー: 「会話で何でも聞ける」
   - できること: ChatGPTやGeminiのようにチャットで会話
   - 特徴: 動くグラフをチャット内に埋め込める
   - イメージ: 「物知りな相談相手」

3. ツール②: Claude Code（AIエージェント）
   - アイコン: ⚡ 稲妻
   - キャッチコピー: 「代わりに作業してくれる」
   - できること: コードを書いて自律的に作業
   - 特徴: 調査、ファイル操作、Web制作、アプリ開発まで
   - イメージ: 「スーパーエンジニアな同僚」

4. ツール③: Cowork（全自動エージェント）
   - アイコン: 🚀 ロケット
   - キャッチコピー: 「指示したら完成まで全自動」
   - できること: タスクを丸投げ→成果物まで一気に完成
   - 特徴: 途中の承認なしでやり切ってくれる
   - 注意: まだ研究段階（research preview）
   - イメージ: 「任せきりにできる助っ人」

5. まとめブロック
   - 3つのツールの関係を矢印で表現:
     Claude（会話）→ Claude Code（作業）→ Cowork（全自動）
   - 「自動化のレベルが上がっていく」ことが視覚的に分かるようにしてください

6. フッター
   - 「まずはClaudeで会話、慣れたらClaude Codeに挑戦！」
   - 公式サイト: claude.ai

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、アクセント #F5F0EB（ライトベージュ）
- 3つのツールは横並びまたはカード形式で、視覚的に比較しやすくしてください
- 各カードは同じレイアウト構造にして統一感を出してください
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 装飾は最小限。各ツールのアイコンで差別化してください
```
