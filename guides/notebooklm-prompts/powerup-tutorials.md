# NotebookLM インフォグラフィック作成プロンプト: 公式チュートリアル /powerup

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ランドスケープ（横長）— 4つのチュートリアルを横並びカードで比較
- **詳細度**: Standard（標準）
- **スタイル**: Kawaii（学習コンテンツに親しみやすい雰囲気）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】/powerup で学べる4つの公式チュートリアル

【対象者】Claude Code を使い始めた初心者〜中級者。次のステップとして何を学べるかを一覧で見せたい。

【構成】ヘッダー、4つのチュートリアルカード、おすすめブロック、フッターの7ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「/powerup でレベルアップ！」
   - サブタイトル: Claude Code 公式チュートリアル一覧
   - 起動方法: /powerup と入力 → ↑↓で選んでEnter
   - コマンド部分はダーク背景のコードブロック風に表示

2. チュートリアル①: Steer with modes
   - アイコン: 🎮 コントローラー
   - 内容: モードを切り替える方法を学ぶ
   - できること: 自動実行モード、プラン作成モードなど
   - レベル: 中級者向け

3. チュートリアル②: Teach Claude your rules
   - アイコン: 📝 メモ
   - 内容: 覚えておいてほしいことを設定する
   - できること: 「日本語で返答して」など毎回言わなくてOKに
   - レベル: 初心者〜中級者向け

4. チュートリアル③: Automate your workflow
   - アイコン: ⚡ 自動化
   - 内容: オリジナルコマンドを登録する
   - できること: よく使う指示を「/自分だけのコマンド」に
   - レベル: 中級者向け

5. チュートリアル④: Code from anywhere
   - アイコン: 📱 スマホ
   - 内容: スマホとClaude Codeを連携する
   - できること: 外出先からもClaude Codeに指示できる
   - レベル: 中級者向け

6. おすすめブロック（特に目立たせる）
   - 星マーク⭐で囲んで強調
   - 「おすすめは Teach Claude your rules！」
   - 「まずはこれを試してみましょう」
   - 背景色を他と差別化して目を引くデザインに

7. フッター
   - 「Claude Code を起動して /powerup と入力してみよう！」

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、背景 #F5F0EB（ライトベージュ）、おすすめ強調 #FFD700（ゴールド）
- 4つのチュートリアルはカード形式で並べ、各カードにアイコンとレベル表示を付ける
- 各カードは同じレイアウト構造（アイコン、タイトル、内容、レベル）で統一感を出す
- おすすめブロックはゴールドの枠線や背景で他と差別化
- コマンド部分はダーク背景（#1A1A2E）にライト文字でターミナル風に表示
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
```
