# NotebookLM インフォグラフィック作成プロンプト: 安心＆便利な設定とセキュリティ対策

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— ステップを上から順に読み進める構成
- **詳細度**: Detailed（詳細）— 設定手順が多いため詳細モードで
- **スタイル**: Instructional（手順ガイドに最適）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code を安全に使うための4つの設定

【対象者】Claude Code 初心者。コピペで設定できる安心感を伝えつつ、セキュリティの大切さを理解してもらう。

【構成】ヘッダー、5つのコンテンツブロック、フッターの7ブロック構成。縦長1枚に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「安全設定ガイド」
   - サブタイトル: 4つの対策でClaude Codeを安心に使おう
   - 補足: 順番にコピペするだけでOK！

2. 🔧 設定: No Flickerモード（快適設定）
   - 見出し: 「画面のちらつきを無くす」
   - 背景色: 青系（快適さを表す色）
   - 問題: 初期状態だと文字表示がチカチカする
   - 解決: チャットアプリのようにスムーズ表示に
   - コマンド: CLAUDE_CODE_NO_FLICKER=1 claude
   - コマンド部分はダーク背景のコードブロック風に表示
   - 「コピペしてEnterするだけ！」と補足

3. 🛡️ 対策①: 作業フォルダを限定する
   - 見出し: 「専用フォルダで起動する」
   - 背景色: 緑系（安全を表す色）
   - やること: 新規フォルダを作って、その中で起動する
   - 理由: デスクトップ全体で起動すると関係ないファイルまで触れてしまう
   - ポイント: 「これだけで安全性がグッと上がる！」を強調

4. 🛡️ 対策②: 許可画面をちゃんと読む
   - 見出し: 「Yes連打は危険！」
   - 背景色: オレンジ系（注意を表す色）
   - ルール1: 中身を確認してからYesを押す
   - ルール2: 「Always allow」は慎重に（今回だけ→Yes、ずっと→Always allow）
   - 補足: 分からないコマンドはClaude Codeに聞いてOK
   - 「慣れないうちは毎回Yesで確認」を推奨マークで強調

5. 🛡️ 対策③: サンドボックスモードを設定
   - 見出し: 「行動範囲を制限する」
   - 背景色: 緑系（安全を表す色）
   - コマンド: /sandbox
   - コマンド部分はダーク背景のコードブロック風に表示
   - 選択肢を2つ表示:
     - auto-allow: 安全なコマンドは自動実行OK
     - regular: 毎回確認してから実行（初心者おすすめ ⭐）
   - 「regularモードがおすすめ！」を目立たせる

6. 💡 豆知識: .envファイルの注意
   - 見出し: 「秘密情報は.gitignoreで守る」
   - 背景色: グレー系（補足情報を表す色）
   - APIキーやパスワードを.envに書くことがある
   - GitHubにアップロードすると全世界に公開されてしまう
   - 対策: Claude Codeに「.envを.gitignoreに追加して」と頼むだけ

7. フッター
   - 「設定完了！これで安心してClaude Codeを使えます」
   - チェックマーク✅で達成感を演出

【デザイン指示】
- カラーパレット: 快適設定 #4A90D9（ブルー）、安全対策 #28A745（グリーン）、注意 #E8A838（オレンジ）、補足 #6C757D（グレー）、背景 #F5F0EB（ライトベージュ）
- 各ブロックを色分けして、設定の種類が一目で分かるようにしてください
- コマンド部分はダーク背景（#1A1A2E）にライト文字でターミナル風に表示
- 各ブロックの左端にステップ番号または盾アイコンを配置して進行を示す
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 「コピペするだけ」「これだけでOK」の安心感を全体のトーンとして保ってください
```
