# NotebookLM インフォグラフィック作成プロンプト: Claude についての説明

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）
- **詳細度**: Standard（標準）
- **スタイル**: Kawaii（教育コンテンツ向け）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude って何？ AIチャットとAIエージェントの違い

【対象者】ITに詳しくない初心者〜中級者。技術用語は避け、親しみやすいトーンで。

【構成】ヘッダー、5つのコンテンツブロック、フッターの7ブロック構成。1画面に収まるようにしてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「Claudeって何ができるの？」
   - サブタイトル: AIチャットとAIエージェントの違いを知ろう

2. AIチャット vs AIエージェント
   - AIチャット（ChatGPT, Gemini, Claude）: 質問すると答えてくれる「サポートセンター」
   - AIエージェント（Claude Code）: 指示したら代わりに作業してくれる「同僚」
   - この違いが分かるアイコンや図解を入れてください

3. Claude で使える3つのツール
   - Claude: チャットで会話。動くグラフも作れる
   - Claude Code: コードを書いて自動で作業してくれるスーパーエンジニア
   - Cowork: 全自動で最初から最後までやり切ってくれる（研究段階）
   - 各ツールにアイコンを付けてください

4. Claude Code の活用例
   - 調べ物をして記事を書いて投稿まで自動化
   - 動画の無音カット、テロップ追加も自動
   - Webページやアプリの作成
   - 「これはゲームチェンジャー！」を強調

5. Claude Code が使える場所
   - ターミナル: 最新機能がいち早く使える
   - VSCode: ファイルの変更が確認しやすい
   - デスクトップアプリ: チャット感覚で使える
   - 3つの比較が分かる簡単な表やアイコンで表現

6. フッター
   - まとめ: 「Claude Code = 代わりに作業してくれる同僚」
   - 公式サイト: claude.ai / code.claude.com

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeのブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、アクセント #F5F0EB（ライトベージュ）
- タイポグラフィ: クリーンなサンセリフ体。ヘッダーはボールド、本文はレギュラー
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内に収めてください
- ホワイトスペースを十分に確保し、読みやすさを最優先にしてください
- 装飾は最小限に。視認性を優先してください
```
