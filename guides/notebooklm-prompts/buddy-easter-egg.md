# NotebookLM インフォグラフィック作成プロンプト: 隠しコマンド /buddy

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: スクエア（正方形）— 内容が少ないのでコンパクトに
- **詳細度**: Concise（簡潔）— お楽しみ要素なので軽く
- **スタイル**: Kawaii（遊び心のある雰囲気にぴったり）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code の隠しコマンドを発見！

【対象者】Claude Code を体験中の参加者。遊び心を刺激して「試してみたい！」と思わせたい。

【構成】ヒーロー、メインブロック、注意、フッターの4ブロック構成。コンパクトに1画面。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「🐣 隠しコマンド発見！」
   - サブタイトル: エイプリルフール企画の名残…？
   - ワクワク感のある演出で

2. メインブロック（中央に大きく）
   - 大きく表示: /buddy
   - コマンドはダーク背景のコードブロック風、特大フォントで
   - 「Claude Codeを起動して、こっそり入力してみよう」
   - 何が起こるかは… お楽しみ！🎁
   - プレゼントボックスや「？」マークで謎めいた演出

3. 注意ブロック（小さめに）
   - ⚠️ 時期によっては終了している場合があります
   - 動かなかったらゴメンなさい！
   - 控えめなグレー文字で

4. フッター
   - 「他にも隠し機能があるかも…？探してみよう！」

【デザイン指示】
- カラーパレット: メイン #FFD700（ゴールド、お宝感）、サブ #D97757（Claudeオレンジ）、背景 #1A1A2E（ダークネイビー、秘密っぽさ）、アクセント #F5F0EB（ライトベージュ）
- 全体をダーク背景にして「秘密のコマンド」感を演出
- /buddy コマンドを中央に大きく、光るようなエフェクト風の装飾で目立たせる
- 🐣ひよこ、🎁プレゼント、✨キラキラのアイコンを散りばめて遊び心を表現
- タイトルは8語以内、箇条書きは10語以内
- ワクワク感と「試してみたい！」を引き出すデザインにしてください
```
