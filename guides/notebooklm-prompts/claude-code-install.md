# NotebookLM インフォグラフィック作成プロンプト: Claude Code のインストール

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: スクエア（正方形）— 内容が少ないのでコンパクトに
- **詳細度**: Concise（簡潔）— ステップが少ないのでシンプルに
- **スタイル**: Instructional（手順ガイドに最適）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code をインストールしよう

【対象者】ターミナル初心者。コマンドをコピペするだけでできる安心感を伝える。

【構成】ヘッダー、3つのステップブロック、フッターの5ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「Claude Code インストール」
   - サブタイトル: コピペするだけ！1分で完了

2. ステップ1: ターミナルを開く
   - アイコン: 🖥️
   - 「作業用フォルダをDockのターミナルにドラッグ＆ドロップ」
   - または「cd フォルダパス」で移動

3. ステップ2: コマンドをコピペして実行
   - コマンド表示: curl -fsSL https://claude.ai/install.sh | bash
   - コマンド部分はモノスペースフォント、ダーク背景のコードブロック風に表示
   - 「このコマンドをそのままコピペしてEnter！」と補足

4. ステップ3: 完了を確認
   - インストールが終わったら「claude」と入力して起動確認
   - 初回はログインが求められるので画面の指示に従う
   - 「↑↓で選択、Spaceで選択、Enterで決定」

5. フッター
   - 参考リンク: Claude Code クイックスタート（code.claude.com）
   - 「困ったらスタッフに声をかけてください！」

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、背景 #F5F0EB（ライトベージュ）
- ステップを1→2→3の番号付きフローで表現し、進行方向が分かる矢印を付けてください
- コマンド部分はダーク背景（#1A1A2E）にライト文字で、ターミナル風に表示
- 「コピペするだけ」の安心感が伝わるよう、シンプルで明快なデザインに
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
```
