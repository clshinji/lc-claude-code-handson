# NotebookLM インフォグラフィック作成プロンプト: Claude Code のセキュリティリスク

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— 危険→安心の流れを上から下へ
- **詳細度**: Standard（標準）
- **スタイル**: Editorial（警告・注意喚起の情報に適した、信頼感のあるデザイン）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】Claude Code を安全に使うために知っておくべき3つのリスク

【対象者】Claude Code 初心者。怖がらせすぎず、でもリスクを正しく理解してもらう。最後に「対策できるから大丈夫！」と安心させる。

【構成】ヘッダー、3つのリスクブロック、安心メッセージ、フッターの6ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「ちょっと待って！その前に」
   - サブタイトル: Claude Code を安全に使うための注意点
   - 補足: 設定なしで「なんでもOK！」は危険かも…

2. リスク①: APIキーやパスワードが漏れる
   - アイコン: 🔑 鍵
   - 危険度を示す赤色の警告マーク
   - 例え話: 秘密の鍵を設定ファイルに保存 → うっかり外部に送信
   - 最悪のケース: 「他人に勝手に使われて高額請求！」
   - 短くまとめて: 秘密情報の保存には要注意

3. リスク②: 大事なファイルを勝手に消される
   - アイコン: 🗑️ ゴミ箱
   - 危険度を示すオレンジ色の警告マーク
   - 例え話: 「フォルダ整理して」→ 大切なファイルが消えた！
   - 短くまとめて: AIの判断が常に正しいとは限らない

4. リスク③: 知らないソフトを勝手にインストール
   - アイコン: ⚙️ 歯車
   - 危険度を示す黄色の警告マーク
   - 例え話: よく分からないまま「OK」連打 → 意図しないソフトだらけに
   - 短くまとめて: 確認せずに許可しない！

5. 安心メッセージ（ポジティブに転換）
   - 背景を緑色系に切り替えて安心感を演出
   - 「怖い！と思った方、安心してください」
   - 「次の手順で安全に使える設定をします！」
   - チェックマーク✅のアイコンで「対策できる」ことを強調

6. フッター
   - 「この後のセキュリティ設定で安全に使えます」

【デザイン指示】
- カラーパレット: 警告部分は赤 #DC3545、オレンジ #E8A838、黄 #FFC107 の3段階。安心部分は緑 #28A745。背景 #F5F0EB（ライトベージュ）
- 3つのリスクは上から「赤→オレンジ→黄」のグラデーションで危険度を視覚化
- 最後の安心メッセージで色調を緑に切り替え、ネガティブ→ポジティブの流れを作ってください
- 各リスクに警告アイコンを付け、一目でリスクだと分かるデザインに
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 怖がらせすぎない程度に注意を引くバランスで
```
