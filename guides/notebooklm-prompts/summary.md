# NotebookLM インフォグラフィック作成プロンプト: ハンズオン会まとめ

## 使い方

1. NotebookLM でノートブックを開く
2. ソースとして `handson-guide.md` をアップロードする
3. **Studio → Infographic** に移動する
4. 以下のプロンプトを貼り付けて生成する

## 設定

- **向き**: ポートレート（縦長）— 振り返りを上から下へ読み進める
- **詳細度**: Standard（標準）
- **スタイル**: Editorial（まとめ・振り返りに適した落ち着いたデザイン）

## プロンプト

```
以下の内容をインフォグラフィックにしてください。

【テーマ】ハンズオン会のまとめ — 今日やったことの振り返り

【対象者】ハンズオン会に参加した初心者〜中級者。達成感を感じてもらい、今後も使い続けたいと思わせたい。

【構成】ヒーロー、2つの振り返りブロック、4つのコツ、次のステップ、フッターの6ブロック構成。1画面に収めてください。

【ブロック内容】

1. ヒーローブロック（タイトル）
   - タイトル: 「お疲れさまでした！」
   - サブタイトル: 今日の体験を振り返ろう
   - 達成感を演出する✨やトロフィー🏆のアイコン

2. 振り返り①: ターミナルでClaude Code
   - アイコン: 🖥️ ターミナル
   - チェックリスト形式（✅付き）で達成感を演出:
     - ✅ インストール → 起動 → 会話 → 終了を体験
     - ✅ セキュリティ対策を設定（フォルダ限定・許可確認・サンドボックス）
     - ✅ /powerup で公式チュートリアルにチャレンジ
   - 全項目にチェックが入った達成感のあるデザイン

3. 振り返り②: デスクトップアプリ
   - アイコン: 💬 アプリ
   - チェックリスト形式:
     - ✅ 「コード」機能を試した
     - ✅ 「Cowork」機能を試した
   - ポイント: ターミナルが苦手でもチャット画面からOK！

4. 今日から使える4つのコツ（横並びまたはアイコン付きリスト）
   - 🎯 指示は具体的に
   - 📋 1つずつお願いする
   - 💬 困ったらClaudeに聞く
   - 🛡️ セキュリティを意識する
   - 4つを横並びのバッジ風デザインで

5. もっと学びたい方へ（次のステップ）
   - 公式ドキュメント: code.claude.com
   - Claude公式サイト: claude.ai
   - 「日々の仕事や趣味にClaude Codeを取り入れよう！」

6. フッター
   - 「今日の体験が、明日のスキルに！」
   - ハンズオン会のロゴやイベント名があれば表示

【デザイン指示】
- カラーパレット: メイン #D97757（Claudeブランドオレンジ）、サブ #1A1A2E（ダークネイビー）、達成色 #28A745（グリーン）、背景 #F5F0EB（ライトベージュ）、アクセント #FFD700（ゴールド）
- チェックリストの✅は緑色で、全て完了した達成感を視覚的に表現
- ヒーローブロックにゴールドのアクセントで「お疲れさま」感を華やかに
- 4つのコツはバッジまたはカード風で、持ち帰れるキーワードとして印象的に
- タイトルは8語以内、ヘッダーは5語以内、箇条書きは10語以内
- ホワイトスペースを十分に確保し、読みやすさを最優先
- 全体のトーンは「やり遂げた！」というポジティブな達成感で統一してください
```
