02 ── PREPARATION

事前に準備してほしいこと

必須

Claude Pro プランに加入する

Claude Codeを利用するには、Claudeの Proプラン（月額 $20）への加入が必要です。

  1. claude.ai にアクセスしてアカウントを作成
  2. 画面左下の「Upgrade」ボタンから Proプランにアップグレード
必須

モデル学習へのオプトアウト設定

入力したプロンプトがAIモデルの学習に使われないようにする設定です。

  1. claude.ai にログイン
  2. 画面左下のアカウント名をクリック
  3. 「設定」を開く
  4. 「プライバシー」タブを選択
  5. 「Claudeの改善に協力する」のトグルをオフにする
なぜオプトアウトが大切？

この設定をしないと、入力したプロンプトやコードがモデルの学習データとして利用される可能性があります。個人情報や業務データを守るために、必ず設定しましょう。

推奨（当日も一緒にやります）

Claude Code をインストールする

当日も一緒に進めますが、事前にやっておくとスムーズです。

  1. Macの「ターミナル」アプリを開く
  2. 以下のコマンドをコピペして実行
$ curl -fsSL https://claude.ai/install.sh | bash
  1. インストールが終わったら claude と入力して起動を確認

公式ガイド: code.claude.com/docs/ja/quickstart

任意

コードエディタをインストールする

必須ではありませんが、あると作業が快適になります。

  1. VS Code ── 無料。Claude Code拡張機能あり
  2. Cursor ── AI機能が組み込まれたエディタ
なぜエディタがあると便利？

Claude Code で Ctrl+G を押すと外部エディタでプロンプトを編集できます。長い指示を書くときにとても便利です。エディタがなくてもMac標準のテキストエディットで代用できるので、当日講師がサポートします。
