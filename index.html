02 ── PREPARATION

事前に準備してほしいこと

当日スムーズに始められるよう、できる範囲で事前にセットアップをお願いします。

必須

Claude Pro プランに加入する

Claude Codeを利用するには、Claudeの Proプラン（月額 $20）への加入が必要です。

  1. claude.ai にアクセスしてアカウントを作成
  2. 画面左下の「Upgrade」ボタンから Proプランにアップグレード
必須

モデル学習へのオプトアウト設定

入力したプロンプトがAIモデルの学習に使われないようにする設定です。

  1. claude.ai にログイン
  2. 画面左下のアカウント名をクリック
  3. 「設定」を開く
  4. 「プライバシー」タブを選択
  5. 「Claudeの改善に協力する」のトグルをオフにする
なぜオプトアウトが大切？

この設定をしないと、入力したプロンプトやコードがモデルの学習データとして利用される可能性があります。個人情報や業務データを守るために、必ず設定しましょう。

推奨（当日も一緒にやります）

Claude Code CLI をインストールする

ターミナル（コマンドライン）から使うClaude Codeです。当日も一緒に進めますが、事前にやっておくとスムーズです。

  1. 「ターミナル」アプリを開く
  2. 以下のコマンドをコピペして実行
$ curl -fsSL https://claude.ai/install.sh | bash
  1. インストールが終わったら claude と入力して起動を確認
  1. 「PowerShell」を開く（スタートメニューで「PowerShell」と検索）
  2. 以下のコマンドをコピペして実行
PS> irm https://claude.ai/install.ps1 | iex
  1. インストールが終わったら claude と入力して起動を確認
Windowsの方へ

事前に Git for Windows のインストールが必要です。

公式ガイド: code.claude.com/docs/ja/quickstart

推奨

Claude デスクトップアプリをインストールする

Claude Codeが組み込まれたデスクトップアプリです。GUIで使えるので、ターミナルに慣れていない方にもおすすめです。

  1. claude.ai にアクセスし「Download」からMac版をダウンロード
  2. ダウンロードした .dmg ファイルを開き、アプリケーションフォルダにドラッグ
  3. アプリを起動し、ログイン後「Code」タブを確認
  1. claude.ai にアクセスし「Download」からWindows版をダウンロード
  2. ダウンロードしたインストーラーを実行
  3. アプリを起動し、ログイン後「Code」タブを確認
任意

コードエディタをインストールする

必須ではありませんが、あると作業が快適になります。

  1. code.visualstudio.com からダウンロード（無料）
  2. インストール後、拡張機能（Cmd+Shift+X / Ctrl+Shift+X）で「Claude Code」を検索してインストール
  1. cursor.com からダウンロード
  2. AI機能が組み込まれたエディタで、すぐに使い始められます
なぜエディタがあると便利？

Claude Code で Ctrl+G を押すと外部エディタでプロンプトを編集できます。長い指示を書くときにとても便利です。
