# 話題の Claude Code 体験してみよう！ ゆるっとハンズオン会

## やりたいこと

- 初心者から中級者までを参加者と想定して、Claude Codeにチャレンジする/使い方をレベルアップすることができるオフ会を開催する

## 作業のルール

- Claude Code の使い方・ノウハウを、参加者に紹介します。 **新しい学びがあったら、ドキュメント化** します！

## ハンズオンで達成したいこと

- まずは、話題の **Claude Code** を体験してみる
- とても便利なツールですが、その分、セキュリティや機密情報の扱いには気をつける必要がある
- 少しずつ試して、あらゆる仕事を効率化できそう、という雰囲気を感じてもらう
