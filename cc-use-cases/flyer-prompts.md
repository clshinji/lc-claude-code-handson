# Claude Code ハンズオン：HTML フライヤーを作ってみよう

このファイルは **コピペで使えるプロンプト集** です
上から順番に Claude Code に貼り付けて、少しずつフライヤーを育てていきます

---

## 事前準備（ターミナルで実行）

新しいフォルダを作って、その中で Claude Code を起動します
以下をターミナルにコピペしてください

```bash
cd ~/Desktop
mkdir flyer-handson
cd flyer-handson
claude
```

起動したら、下のプロンプトを順番に試していきましょう

---

## STEP 1 🎨 最初のフライヤーを作る（まずは1つ完成させる）

まずはこれをそのままコピペして送ってみてください
細かい指定はせず、Claude Code に"お任せ"で作ってもらいます

```
HTMLファイル1枚で完結する、おしゃれなイベント告知フライヤーを作ってください。

【イベント情報】
- タイトル：春のピアノ発表会
- 日時：2026年5月18日（日）14:00〜16:00
- 場所：リベシティホール 3F
- 対象：どなたでもご参加いただけます
- 参加費：無料
- 主催：チャンモカピアノ教室

【デザインの要望】
- A4縦サイズで印刷してもキレイに見えるようにしてください
- 春らしい優しい色合い（淡いピンク・クリーム色・新緑グリーンなど）
- 外部の画像やCSSは読み込まず、1ファイルで完結させてください
- 絵文字やシンプルな図形（CSSで描ける程度）でアクセントをつけてOKです
- ファイル名は index.html にしてください

作り終わったら、ブラウザで開く方法も教えてください。
```

> 💡 ポイント：完成したら、ターミナルに表示される `open index.html` を実行するか、Finder でファイルをダブルクリックするとブラウザで見られます

---

## STEP 2 ✏️ 内容を自分のフライヤーに書き換える

ここからが本番です
**自分の業務・事業に合わせて** 内容を差し替えてみましょう
以下のテンプレをコピペして、【　】の中だけ自分の情報に書き換えてください

```
さっき作った index.html の内容を、下記の情報で作り直してください。
デザインの雰囲気は同じ方向性で、色味だけ内容に合った配色に変えてください。

【イベント・サービス名】
（例：夏休み親子でプログラミング体験会）

【日時 or 期間】
（例：2026年8月3日(土) 10:00〜12:00）

【場所 or オンライン/オフライン】
（例：〇〇市民センター 会議室A）

【対象】
（例：小学3〜6年生とその保護者）

【料金】
（例：親子ペア 2,000円）

【主催・連絡先】
（例：ライオンスクール）

【伝えたい雰囲気】
（例：わくわく楽しそう / 落ち着いた高級感 / ポップで元気 など）
```

## STEP 3 🔁 作ったものを少しずつ調整する（自然言語で修正）

ここが Claude Code の **一番の楽しいところ** です
下記のようにカジュアルに話しかけるだけで、その場でファイルを直してくれます
ピンときたものを1〜2個、試してみましょう

### 色を変えたい

```
全体をもう少し落ち着いた色にしてください。ベージュと深緑を基調にしてほしいです。
```

### 文字サイズ・バランスを整えたい

```
タイトルがちょっと小さく感じます。もう一回り大きくして、日時をより目立たせてください。
```

### 要素を足したい

```
QRコード用の四角い枠を右下に追加してください（中身は空の枠でOKです）。
「詳細はこちら」というキャプションもつけてください。
```

### SNS映え用の別バージョンが欲しい

```
index.html はそのまま残して、同じ内容で Instagram 投稿用の
1080x1080 の正方形バージョンを square.html として新しく作ってください。
```

### 印刷に最適化したい

```
A4印刷したときに余白や改ページが崩れないように、印刷用のCSSを追加してください。
```

## STEP 4 🚀 応用：自分の業務で使えそうな派生物を作る

余裕があればチャレンジしてみてください
"1つのデザインから派生物を量産できる" ことを体感するステップです

```
いま作ったフライヤーと同じデザインテイストで、以下の3つも作ってください。

1. sns.html … Instagram / X 投稿用の正方形バージョン
2. thanks.html … 参加者にお渡しする「ご来場ありがとうございました」カード
3. reminder.html … 前日リマインド用のシンプル告知（最小限の情報だけ）

全部同じフォルダに保存してください。
```
